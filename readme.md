## üèÅ Starting

```bash
# Clone this project
$ git clone https://github.com/timoteostifft/ewally_node.git
# Access
$ cd ./ewally_node
```

### Node

```bash
# Install dependencies
$ yarn
# Run the project
$ yarn dev
# The server will initialize at <http://localhost:8080>
```

### Docker

```bash
# Run project
$ docker-compose up
# The server will initialize at <http://localhost:8080>
```
